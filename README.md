# Nam Tân Uyên — Website

Modern rebuild of the Nam Tân Uyên industrial park corporate website.

## Stack

- Next.js 15 (App Router, SSG)
- TypeScript
- Tailwind CSS 4
- Framer Motion + Lenis
- Content modules in `src/content`

## Develop

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Build

```bash
npm run build
npm start
```

## Legacy source

Original HTML/CSS/JS is preserved in `legacy/` for reference. Old `.html` URLs permanently redirect to the new routes.
